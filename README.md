# Tenjou Studio — Architecture Landing Page

<img src="https://marvinnox.github.io/Jacopo-Marchesoni-porfolio/assets/logo.png" alt="Tenjou Studio logo" width="800">

## 📌 Overview

Tenjou Studio is a modern landing page for an architecture studio focused on **sustainable, nature-inspired design**. The project showcases the studio’s philosophy, services, and visual identity.

The website emphasizes:

- Sustainable architecture
- Wood-based construction
- Passive house design
- Minimalist and Japanese-inspired principles
